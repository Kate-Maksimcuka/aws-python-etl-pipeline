# AWS Python ETL Pipeline

## Project Overview

This beginner data engineering project demonstrates a simple ETL pipeline using Python and AWS-style storage.

The pipeline extracts raw data, cleans and transforms it with Python, saves the cleaned dataset, and produces basic summary charts.

## Project Goals

- Practise building a simple ETL workflow
- Clean and transform raw data using Python
- Store processed data in a structured format
- Create summary outputs and visualisations
- Document the pipeline clearly for a junior data engineering portfolio

## Tools

- Python
- pandas
- AWS S3
- AWS Athena
- matplotlib or Plotly
- GitHub

## Planned Workflow

1. Extract data from a public API or CSV file
2. Clean missing values and standardise columns
3. Save cleaned data as CSV or Parquet
4. Upload processed data to S3
5. Query the cleaned data using Athena
6. Create simple charts from the final dataset

## Planned Outputs

- Cleaned dataset
- ETL script
- Basic logging or status messages
- SQL query examples
- 3–5 graphs
- Architecture diagram

## Example Charts

- Records over time
- Category totals
- Average value by category
- Top 10 items or locations
- Missing data before/after cleaning

## What This Project Demonstrates

- Python scripting
- Data cleaning
- ETL thinking
- Basic AWS data workflow
- Clear project documentation
